Privacy Policy
Last Updated: March 21, 2026
This Privacy Policy describes how the Fluxer bot (referred to as “Functious”) collects, uses, stores, and deletes data. Functious operates exclusively on the Fluxer platform. This policy applies only to data handled by Functious. Fluxer itself has its own separate privacy policy. By using Functious, you agree to the practices outlined below.

1. Data We Collect
We only collect the absolute minimum data required for Functious’ features to function. No messages, usernames, emails, IP addresses, analytics data, or any other personal information is stored. We do not use cookies, tracking pixels, or any form of user tracking.

User IDs (Fluxer snowflake IDs)
Giveaways: Only the User IDs of people who enter a giveaway.
Polls: User IDs and publicly available avatar URLs fetched from Fluxer (to show who voted for each option).
Timezone system: User IDs paired with the timezone you set.

Guild IDs (Fluxer server IDs)
Stored for every server Functious is in, so it can remember per-server custom settings and command configurations.
Temporary Voice Channels
Only User IDs of people who join voice channels created by Functious’ temporary channel feature.

2. How We Use the Data

Giveaway User IDs → to track entries and pick winners.
Poll User IDs + avatar URLs → to display accurate vote results in the poll embed.
Timezone User IDs → to convert times correctly between users.
Guild IDs → to apply server-specific settings (prefixes, roles, channels, etc.).
Voice User IDs → to manage temporary voice channels (e.g., creating or deleting them based on who is present).

3. Data Retention & Automatic Deletion
All data is deleted as soon as it is no longer needed:

Giveaway User IDs → automatically removed the moment the giveaway ends.
Poll User IDs & avatar URLs → automatically removed the moment the poll ends.
Temporary Voice Channel User IDs → automatically removed the instant you leave the channel.
Guild settings → kept only while Functious is still in the server (they are cleared if Functious leaves or settings are reset).

4. Your Right to Delete Your Data

Timezone data: You can remove your own User ID and timezone at any time by using Functious’ dedicated removal command (exact command name is shown in Functious’ help menu).
All other data (giveaways, polls, temporary voice) is automatically deleted on schedule — no manual action is required.

5. Data Storage & Security
Data is stored in a secure database hosted locally on home lab hardware. Nothing is stored externally — no cloud providers, no third-party servers, and no data leaves the developer’s physical equipment. We use industry-standard security practices (encryption at rest, access controls, etc.), but no system is 100% secure.

6. Children’s Privacy
Functious is not directed at children under 13. If you are under 13, please do not use Functious. We do not knowingly collect any data from children under 13.

8. No Sharing with Third Parties
We do not sell, rent, or share any of this data with anyone. Functious does not send data to external services except for what Fluxer’s own API requires to operate (e.g., fetching avatar URLs or managing voice channels).

10. Changes to This Policy
We may update this Privacy Policy occasionally. The “Last Updated” date at the top will change, and we will announce major updates in Functious’ official support server.

12. Questions or Concerns
If you have any questions about this Privacy Policy or want confirmation that your data has been deleted, you can:

Use Functious’ built-in commands (e.g., timezone removal), or
Contact the developer directly through the official support server (link is available in Functious’ bio or info command).

By using Functious, you acknowledge that you have read and understood this Privacy Policy.
